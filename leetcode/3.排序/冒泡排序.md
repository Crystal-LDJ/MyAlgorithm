## 冒泡排序
```
    冒泡排序是最慢的排序算法之一，但也是最容易实现的算法之一
		之所以叫冒泡排序是因为这种排序方式，数据值会像气泡一样从数组的一端漂浮到另外一端。
		假设正在将一组数字按照升序排序，较大的值会浮动到数组的右侧，而较小的值则会浮动到数组的左侧。
		产生这种现象是因为算法会多次在数组中移动，比较相邻的数据，当左侧数值>右侧数值就会将他们互换		
```

### 示例:
```
    输入: [2,1,8,9,6,17]
    输出: [1,2,6,8,9,17] 
```

### 解题思路

![冒泡](/bubble.gif)
```
1.比较相邻的两个数值，如果前>后 则互换
2.不断对比 直到与最后一个数值比较，那么最大的数值就放在最后，确认最大数值
3.重复 12操作，继续不断对比相邻数值，上次确认的最大数值，直到排序成功
```

### 代码实现
```
   function bubbleSort(arr){
		 var length = arr.length
		 for(let i = 0;i<length;i++){
			 for(let j = 0; j<length-i-1;j++){			 
				 if(arr[j]>arr[j+1]){
					 let temp = arr[j+1];
					 arr[j+1] =  arr[j];
					 arr[j] = temp;
				 }		 
			 }
		 }
		 return arr
	 }
```
